About Riff

Welcome to Riff, your lightweight, context-aware personal snippet sanctuary. Unlike bloated productivity suites that require massive browser extensions and constantly capture your typing, Riff is built to provide you with complete control, speed, and privacy. It serves as a clean offline playground and a robust cloud vault for all your code blocks, recipes, bookmarks, thoughts, and technical credentials.

Two Modes of Operation

I believe in giving you absolute autonomy. Riff is engineered to support two distinct operating flows:

Guest Mode (100% Offline)

Perfect for local-only testing, offline sandboxing, or complete network isolation.

  • Stored entirely on your device's browser memory (localStorage).
  • Zero sync network payload — completely private to your PC.
  • Allows rapid one-click migration to secure cloud sync anytime.

Cloud Sync (Authenticated)

Best for developers using multiple machines who need instant, secure code synchronization.

  • Powered by premium serverless database tech (Convex Cloud).
  • Cryptographically signed session tokens for maximum security.
  • Features user-photo cropping, GDPR data backup, and session logs.

Custom Core Features

A
Adaptive Search & Categorization: Instantly filter dozens of code blocks by dynamic category tabs (General, Recipes, bookmarks, Code snippets, etc.) or look up content in real time with our client-side fuzzy-find index.
B
Advanced Account Management: Track your login location and browser footprint in our transparent session logger. Revoke older sessions, change secure passwords, or purge your profile instantly with a single button.
C
GDPR Compliance: Get direct, one-click export of everything you store. We package your custom categories, profile detail history, and snippets into a clean, raw JSON schema instantly.

My Architectural Pillars

Riff is crafted by Rohit utilizing a clean, highly modern tech stack that values efficiency:

Frontend: HTML5 Semantic DOM + Vanilla Javascript ES6 + TailwindCSS

Backend: Convex Cloud Infrastructure (Serverless Queries, Mutations, and Reactive Listeners)

Authentication: Custom Cryptographic Hashed Session Logs

Data Sovereignty: Client-side LocalStorage sandboxing (Zero server logging on local use)
